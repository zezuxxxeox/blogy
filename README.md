# BLOGY

사진, 원하는 블로그 글 레퍼런스, 프롬프트를 넣으면 사진의 맥락과 글 섹션을 자동으로 맞춰서 블로그 에디터에 바로 붙여넣을 수 있는 결과물을 만드는 로컬 웹앱입니다.

## 실행

바탕화면의 `BLOGY` 바로가기를 더블클릭하면 로컬 서버가 자동으로 켜지고 독립 앱 창처럼 열립니다.

Microsoft Edge가 설치되어 있으면 주소창 없는 앱 모드로 열리고, Edge를 찾지 못하면 기본 브라우저로 열립니다.

바로가기를 다시 만들어야 할 때는 프로젝트 폴더에서 아래 명령을 실행하면 됩니다.

```powershell
npm run shortcut
```

Node.js가 설치되어 있어야 합니다.

## 앱처럼 설치하기 (PWA)

BLOGY는 설치형 웹앱(PWA)입니다. 앱스토어 없이, 무료로, 휴대폰과 PC 홈 화면에 아이콘으로 설치할 수 있습니다.

### PC (Windows / Mac)

1. BLOGY를 Edge 또는 Chrome으로 엽니다. (바탕화면 `BLOGY` 바로가기로 열어도 됩니다.)
2. 주소창 오른쪽 끝의 **설치 아이콘**(모니터 + 아래 화살표 모양)을 누릅니다.
   - 또는 메뉴 → `앱` → `이 사이트를 앱으로 설치`.
3. 설치하면 독립 창과 시작 메뉴 아이콘이 생기고, 일반 프로그램처럼 켤 수 있습니다.

### 휴대폰 (iPhone / Android)

휴대폰에서 쓰려면 먼저 아래 "무료로 웹에 올리기"로 인터넷 주소를 만들어야 합니다. 그 주소를 휴대폰 브라우저로 연 다음:

- **iPhone (Safari):** 공유 버튼 → `홈 화면에 추가`
- **Android (Chrome):** 메뉴(⋮) → `앱 설치` 또는 `홈 화면에 추가`

설치하면 주소창 없는 앱처럼 전체 화면으로 열리고, 인터넷이 끊겨도 화면은 뜹니다. (AI 생성은 인터넷이 필요합니다.)

## 무료로 웹에 올리기

BLOGY는 서버 없이 브라우저 안에서만 동작하는 정적 웹앱이라, 무료 정적 호스팅에 폴더만 올리면 어디서나 주소로 접속됩니다. 셋 중 아무거나 쓰면 됩니다. 모두 무료이고 HTTPS가 자동으로 적용됩니다(PWA 설치에 HTTPS가 필요함).

| 서비스 | 올리는 방법 | 가격 |
| --- | --- | --- |
| **Netlify Drop** | `app.netlify.com/drop` 에 이 폴더를 통째로 끌어다 놓기 | 무료 |
| **Cloudflare Pages** | 대시보드에서 폴더 업로드 또는 GitHub 연결 | 무료 |
| **GitHub Pages** | 이 폴더를 GitHub 저장소에 올리고 Settings → Pages 활성화 | 무료 |

올릴 때 `server.mjs`, `Start-*.ps1`, `*.vbs`, `*.cmd`, `.server*.log`는 빼도 됩니다. 실제로 필요한 파일은 `index.html`, `app.js`, `styles.css`, `manifest.webmanifest`, `service-worker.js`, `assets/` 안의 사용 중인 이미지입니다.

올린 뒤 받은 주소(예: `https://blogy.netlify.app`)를 휴대폰/PC 브라우저로 열고 위의 "앱처럼 설치하기"를 따라 하면 됩니다.

> 앱을 고쳤을 때는 `index.html`의 `?v=9`, `service-worker.js`의 `CACHE_VERSION` 숫자를 함께 올려야 설치된 기기에 새 버전이 반영됩니다.

### 진짜 앱스토어에 올리고 싶다면

iOS 앱스토어·구글 플레이 정식 등록도 가능하지만, 비용과 절차가 듭니다. 이 PWA를 [Capacitor](https://capacitorjs.com/)로 감싸면 됩니다. 다만 Apple 개발자 등록 연 $99, 구글 플레이 1회 $25, 그리고 심사 과정이 필요합니다. 대부분의 경우 위의 무료 PWA 설치로 충분합니다.

## 무료 API 정책

- 기본 고품질 모드는 Gemini API 무료 티어를 사용합니다. 앱 안의 `무료 키 만들기` 링크에서 Google AI Studio 키를 만들고 붙여넣으면 됩니다.
- API 키는 서버로 저장하지 않고 현재 브라우저의 `localStorage`에만 저장합니다.
- Gemini 호출이 실패하면 임시 로컬 결과로 대체됩니다. 고품질 결과는 Gemini 키 입력을 기준으로 합니다.
- 블로그 자동 업로드 버튼까지 누르는 기능은 플랫폼별 약관과 로그인 자동화 이슈가 있어 넣지 않았습니다. 대신 네이버/티스토리/워드프레스 편집기에 붙여넣기 좋은 HTML과 일반 텍스트를 만듭니다.

## 추천 사용 흐름

1. 사진을 여러 장 업로드합니다.
2. 사진별 메모에 장소, 메뉴명, 제품명, 장면을 한두 단어로 적으면 사진 배치 품질이 좋아집니다.
3. 원하는 글 프롬프트와 참고 블로그 글을 붙여넣습니다.
4. `작성하기`를 누릅니다.
5. `사진 배치`에서 어긋난 사진이 있으면 섹션을 직접 바꿉니다.
6. `블로그용 복사`를 누르고 블로그 에디터에 붙여넣은 뒤 업로드만 누르면 됩니다.
